A simple mathematical cheat-sheet to help with tracking the math when experimenting with transformer-based AI models using PyTorch or similar frameworks.

The motivation here is to condense the core mathematical concepts while trying to define as many concepts and terms as fully as possible (especially matrix dimensions) so that the reader can follow exactly how the calculations are being performed in such architectures. 
The succinctness of the mathematical equations (if fully defined and explained) often helps with understanding implementation details and possibilities for further enhancements within actual code.
